Einleitung
Herzlich willkommen zur Ransomware Demonstration von Studenten für Studenten. Nun ist es endlich soweit und Sie können die Ransomware unter dem Menüpunkt „Demo starten“ auf Ihren eigenen Rechner zum Laufen bringen. Sie werden dabei von „Sixbi“, unserer virtuellen interaktiven Lernumgebung, in die Demonstration eingeführt. Diese erklärt Ihnen jede einzelne Komponente in dem Demonstrationsfenster. Bevor Sie mit der Demonstration beginnen, können Sie sich die restlichen Informationen auf dieser Seite durchlesen.
Was passiert mit meinem Computer?
Zu aller erst muss erwähnt werden, dass die Ransomware keinen Einfluss auf Ihre persönlichen Daten hat. Eine von uns kreierte Bash Datei durchsucht Ihren Computer nach dem Ordner „meinSehrWichtigerOrdner“, welcher im Idealfall auf Ihren Desktop liegen sollte. Die darin enthaltenen Daten werden dann mit dem sogenannten AES-Verschlüsselungsalgorithmus verschlüsselt.
Kann ich die Daten wieder entschlüsseln?
Natürlich. Der von uns generierte Schlüssel, der zum Verschlüsseln benutzt wurde, wird auch zum Entschlüsseln verwendet. Geben Sie dafür während der Demonstration in das Eingabefeld „BitteEntsperren!“ ein und alle Dateien aus dem Ordner „meinSehrWichtigerOrdner“ werden wieder entschlüsselt und sind wieder verfügbar. Außerdem haben Sie die Möglichkeit, die Demonstration durch den Knopf „Zurück“ abzubrechen, dadurch werden die Dateien ebenfalls wieder entschlüsselt.
Was kann ich tun wenn ich die Daten aus dem Ordner „meinSehrWichtigerOrdner“ ausversehen gelöscht habe bzw. die Daten unbrauchbar sind?
Das ist kein Problem. In dem von uns bereitgestellten Ordner „Backup“ befindet sich eine Kopie der Daten. Kopieren Sie diese einfach in den Ordner „meinSehrWichtigerOrdner“.
